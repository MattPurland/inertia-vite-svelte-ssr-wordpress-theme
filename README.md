## WordPress/Vue Starter Theme Using Inertia.js

A bare-bones example theme using Inertia with Vue, and the WordPress Inertia adapter.

### Installation
Clone this repository to WordPress themes directory.

```bash
cd *your-theme-name*

composer install

npm install
npm run dev
```
